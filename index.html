IIS配置教程

发表于 2021-03-10 更新于 2021-04-03 分类于 服务器

IIS简述

IIS 是 Miscrosoft 公司旗下产品，最新版为 IIS 10.0 ，正版 Windwos10 自带该应用。IIS 是 Windows 平台上的 Web 服务器软件，支持 PHP 、 Python 等语言。启用 IIS 方法，第一步打开控制面板：打开 “运行” 【Win键 + R键】 输入 “control” 并按下回车；第二步打开 Windows功能：将控制面板查看方式切换为 “类别” 并打开 “程序” 选择子选项 “启用或关闭 Windows 功能”；第三步启用相关组件：选择所有 “.NET” 开头的相关组件以及 “Internet Information Services” 开头的相关组件然后点击确认，等待安装完成后重启电脑即可完成安装。

程序设计入门

发表于 2021-01-01 更新于 2021-04-03 分类于 算法竞赛入门经典 ， 语言篇

学习目标

  • 熟悉C语言程序编译和运行
  • 学会编程计算并输出常见的算术表达式的结果
  • 掌握整数和浮点数的含义和输出方法
  • 掌握数学函数的使用方法
  • 初步了解变量的含义
  • 掌握整数和浮点数变量的声明方法
  • 掌握整数和浮点数变量的读入方法
  • 掌握变量交换的三变量法
  • 理解算法竞赛中的程序三步曲：输入、计算、输出
  • 记住算法竞赛的目标及其对程序的要求

计算机速度快，很适合做计算和逻辑判断工作。本章首先介绍顺序结构程序设计，其基本思路是：把需要计算机完成的工作分成若干个步骤，然后依次让计算机执行。请注意这里的“依次”二字——步骤之间是由先后顺序的。这部分的重点在于计算。

接下来介绍分支结构程序设计，用到了逻辑判断，根据不同情况执行不同语句。本章内容不复杂，但是不容忽视。

注意：编程不是看会的，也不是听会的，而是练会的，所以应尽量在计算机旁阅览本书，以便把书中的程序输入到计算机中进行调试，顺便再做做上机练习。千万不要图快——如果没有足够的时间用来实践，那么学得快，忘得也快。
