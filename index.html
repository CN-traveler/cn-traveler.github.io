学习目标

  • 熟悉C语言程序编译和运行
  • 学会编程计算并输出常见的算术表达式的结果
  • 掌握整数和浮点数的含义和输出方法

《算法竞赛入门经典》
编程不是看会的，也不是听会的，而是练会的，所以应尽量在计算机旁阅读本书，以便把书中的程序输入到计算机中进行调试，顺便再做做上机练习。千万不要图快——如果没有足够的时间来实践，那么学得快，忘得也快。

软件危机

  • 定义
    软件危机是指计算机软件的开发与维护过程中遇到的一系列严重问题。
  • 表现
  • 原因
    • 客观
      软件特点-逻辑、复杂、手工、维护、成本高
    • 主观
      开发者的认知-需求分析、软件配置（软件开发过程）、重视软件维护
  • 消除途径
    • 技术
    • 管理
    • 正确认识

软件工程

  • 定义
    软件工程是一门解决软件危机（软件开发与维护）的工程学科。【从技术与管理两个角度指导软件的开发与维护】采用工程的概念、原理、技术和方法来开发与维护软件，把经过时间考验而证明正确的管理技术和当前能够得到的最好的技术方法结合起来，以经济地开发出高质量的软件并有效的维护它。
  • 基本目标
    • 满足用户需求-功能、性能等
    • 及时交付-提高生产率
    • 不超过预算-合理预算
    • 不故障-提高可维护性
  • 本质特性
    • 软件工程关注于大型程序的构造
    • 软件工程的中心课题是控制复杂性
    • 软件经常变化
    • 开发软件的效率非常重要
    • 和谐地合作是开发软件的关键
    • 软件必须有效地支持它的用户
    • 在软件工程领域中通常由具有一种文化背景的人替具有另一种文化背景的人创造产品
  • 基本原理
    • 分阶段
      严格管理
    • 阶段评审
    • 严格的产品控制
    • 现代程序设计技术
    • 结果能清楚审查
    • 人员少且精
    • 改进软件工程实践

IIS配置教程

发表于 2021-03-10 更新于 2021-03-26 分类于 教程

IIS简述

IIS 是 Miscrosoft 公司旗下产品，最新版为 IIS 10.0 ，正版 Windwos10 自带该应用。IIS 是 Windows 平台上的 Web 服务器软件，支持 PHP 、 Python 等语言。启用 IIS 方法，第一步打开控制面板：打开 “运行” 【Win键 + R键】 输入 “control” 并按下回车；第二步打开 Windows功能：将控制面板查看方式切换为 “类别” 并打开 “程序” 选择子选项 “启用或关闭 Windows 功能”；第三步启用相关组件：选择所有 “.NET” 开头的相关组件以及 “Internet Information Services” 开头的相关组件然后点击确认，等待安装完成后重启电脑即可完成安装。

IIS网站

IIS 为 Web 服务器软件，所以其主要功能为架设 Web 网站。创建网站方法，第一步打开 IIS ：打开 “运行” 【Win键 + R键】 输入 “inetmgr” 并按下回车；第二步创建 Web 网站目录：在想要创建 Web 网站的文件夹或驱动器下新建文件夹名称自定，创建之后将 Web 网站文件导入该文件夹，然后复制该文件夹绝对路径；第三步创建 Web 站点：在 IIS 管理器中左侧为服务器结构树，在该树中展开你所在服务器的树结构，然后可以看到服务器名称结构至少包含两个子结构 “应用程序池” 、 “网站” ，将鼠标挪至 “应用” 结构上并右键点击 “网站” 然后在右键菜单中选择 “添加网站……” 配置网址名称【名称自定】、应用程序池【？】、物理路径【粘贴之前复制的 Web 网站文件夹路径】、选择通信协议【http协议、https协议（需配置ssl证书）】、绑定IP地址【绑定公网IP可在公网访问、绑定内网IP可在内网访问】、配置主机名【域名】后点击确定，即可完成新建 IIS 的 Web 网站及其的基本配置。
